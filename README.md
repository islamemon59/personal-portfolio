# Personal Portfolio

A modern Next.js + TypeScript portfolio based on the provided plan document, with GSAP-powered scroll animation and interactive sections.

## Run Locally

```bash
npm install
npm run dev
```

Replace the demo content in `src/data/portfolio.ts`, the images in `public/assets`, and the resume at `public/resume.pdf`.

## Contact Email

Copy `.env.example` to `.env.local` and add your SMTP details. For Gmail, use an app password rather than your normal account password.
